NOTE: This tool has only only been tested in Firefox and Chrome.

This demonstration NCI-META terminology maintenance environment. It is loaded with a sample data set derived from the 201604 NCI Metathesaurus. It contains portions of many terminologies, including SNOMED CT, MeSH, ICD10, RXNORM, and others. The data set contains about 2,200 concepts with about 19,000 individual atoms from constituent terminologies.

The environment is configured with users from Jekyll. To log in use your Jekyll initials (e.g. BAC) as the username and password.

Currently the full 201610 NCI Metathesaurus is being loaded and will soon be deployed.
